GET /api/instrument/{symbol}/recommend?timeframe=15
The analysis endpoint produces a TrendStateResult and the recommend endpoint produces a SignalResult. There are serious bugs in both. Fix everything listed below.

PART 1 — Analysis Endpoint Fixes

5. Deduplicate and validate reasoning points
Find where reasoningPoints list is built. It currently concatenates points from multiple stages causing duplicates and contradictions. Fix:
csharp// After building the full reasoningPoints list, deduplicate by topic:
// Only one RSI point, one MACD point, one EMA point, one ADX point per response.

    .GroupBy(r => ExtractIndicatorTopic(r))  // group by which indicator it's about

// Helper to identify which indicator a reasoning string is about:
private static string ExtractIndicatorTopic(string reason)
{
    if (reason.Contains("RSI"))       return "RSI";
    if (reason.Contains("MACD"))      return "MACD";
    if (reason.Contains("EMA"))       return "EMA";
    if (reason.Contains("ADX"))       return "ADX";
    if (reason.Contains("VWAP"))      return "VWAP";
    if (reason.Contains("Bollinger")) return "Bollinger";
    if (reason.Contains("Volume"))    return "Volume";
    return reason;
}

PART 3 — Summary Of All Gates (implement as a single validation method)
Create a method ValidateSignalGates that runs all checks in order and returns the first failure reason:
csharpprivate static string? ValidateSignalGates(
    TrendStateResult analysis,
    IndicatorValues indicators,
    decimal lastClose,
    string direction)
{
    if (analysis.EntryGuidance == null)
        return "No entry guidance from analysis";

    if (analysis.Confidence < 65)
        return $"Confidence {analysis.Confidence} below minimum 65";

    if (analysis.TrendState.State != "PULLBACK_READY" 
        && analysis.TrendState.State != "BREAKOUT_READY")
        return $"State {analysis.TrendState.State} not actionable";

    if (indicators.ADX < 25)
        return $"ADX {indicators.ADX:F1} below minimum 25 — trend not confirmed";

    if (indicators.ADX > 60)
        return $"ADX {indicators.ADX:F1} above 60 — trend exhaustion risk";

    if (direction == "BUY"  && lastClose < indicators.VWAP)
        return $"Price {lastClose} below VWAP {indicators.VWAP} — no buy signal";

    if (direction == "SELL" && lastClose > indicators.VWAP)
        return $"Price {lastClose} above VWAP {indicators.VWAP} — no sell signal";

    if (!IsMarketOpen())
        return "Market is closed";

    return null;  // all gates passed
}

// Usage:
var gateFailure = ValidateSignalGates(analysis, indicators, lastClose, direction);
if (gateFailure != null)
{
    _logger.LogInformation("Signal blocked for {Symbol}: {Reason}", symbol, gateFailure);
    return new NoSignalResult { Reason = gateFailure };
}
